CS648 Assignment 6

JavaScript Conditional Logic and Looping Problems

Instructions:

Click the buttons below to run each problem individually. Make sure to open the browser's developer console (F12) to see the output for problems that use console.log().

1. Larger or Smaller (5 points)

Create an application that accepts two integers within two separate prompts. Then, display only the larger of the two within the browser window.

2. The "Coin Flip" Game (10 points)

Interactive coin flip game where you select "Heads" or "Tails" and see if you win or lose!

3. The "Coin Flip" Game Redux (10 points)

Coin flip game using a for loop that runs 10 times. Check the console for output.

4. The "Coin Flip Streak" Game (5 points)

See how many "Heads" you can get in a row! Check the console for output.

5. Looping a Triangle (5 points)

Display a triangle pattern using loops. Check the console for output.

6. Odd or Even? (5 points)

Check if numbers from 0 to 15 are odd or even. Check the console for output.

7. Marco! Polo! (5 points)

FizzBuzz-style problem with "Marco!" and "Polo!". Check the console for output.

8. Countdown (5 points)

Count down from a user-entered number to 0. Check the console for output.

Note: To see the console output for problems 3-8, open the browser's developer tools (F12) and go to the Console tab.
